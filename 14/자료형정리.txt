js에서 자료형은 따로 선언하지 않는다

숫자형 var num = 100 || -999  (정수형 & 실수형 포함)
문자형  var string = "aa" || 'aa'
논리형  var boolean = true || false
배열    var arr = ['a', 'b' ,'c']
객체    var data = new Date()

자료형을 알아보기 위해선 typeof 함수를 선언한다.


undeined나 null은 필요에 따라 사용한다.

배열은 빈 배열도 선언 가능

js는 할당 연산자 사용가능
+= -= *= /= %= 

연산자 관련해서는 자바와 비슷하다고 생각해도 될 것 같다.
