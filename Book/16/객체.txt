js에서 객체는 프로그램에서 인식할 수 있는 모든 대상을 가리킨다라는 정도로 이해

웹과 관련된 대상을 모두객체로 인식

문서 객체 모델 (DOM) : 웹 문서 자체도 객체이고 그안에 삽입 되어 있는 이미지와 링크, 텍스트 필드 등도 모두 객체
일반적으로 웹 문서에삽입하는 요소는 doucnment, image, link 객체 등이 있다

브라우저 관련 객체 : 웹 브라우저에서 사용하는 정보도 객체로 나타낼 수 있다. 사용하는 브라우저 정보를 담고 있는 navigator 객체를 비롯해 history, location, screen 객체 등이 있다

내장 객체 : 웹 프로그래밍을 할 때 자주 사용하는 요소는 자바스크립트 안에 미리 객체로 정의되어 잇는데 이를 내장 객체라고 한다. 예를 들어 날자, 시간과 관련된 프로그램을 개발하려면 Date 객체를 가져와 쉽게 사용 가능

객체 선언 방식

var 변수명 = new 객체()

객체에는 프로퍼티 property와 메서드 method가 있다
         (특징 || 속성)       (객체가 할 수 있는 동작)       


Array 객체의 메서드가 은근 많다 필요할 때마다 참고하면서 익히자
